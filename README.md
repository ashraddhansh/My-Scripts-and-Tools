= My Scripts and Tools

== `book`
Opens an `fzf` interface to search and open books stored in a specified directory.

== `compress-pdf`
A CLI utility for compressing PDFs locally. I prefer avoiding online tools for privacy reasons, and I like living in the terminal.

== `csv-to-html-table.py`
Converts a CSV file into an HTML table.

== `cwall`
A utility for changing wallpapers in Hyprland.

== `define.sh`
A quick dictionary lookup tool. Pass a word as an argument and it fetches the definition using the #[api.dictionaryapi.dev](="https://api.dictionaryapi.dev") API.

== `outline`
Extracts and displays the outline or table of contents from a PDF or book.

== `table-to-csv`
Converts HTML tables into CSV format.
